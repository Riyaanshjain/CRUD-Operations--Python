# Python CLI File Manager

A simple command-line file manager built using Python.
It allows users to perform basic CRUD operations on files in the current directory.

## Features

* Create files
* Read files
* Update files

  * Append content
  * Rename files
  * Overwrite files
* Delete files

## Technologies Used

* Python
* pathlib

## How to Run

```bash
python main.py
```

## Example Menu

```
1 Create file
2 Read file
3 Update file
4 Delete file
5 Exit
```
